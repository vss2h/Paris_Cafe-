# Paris Cafe
Project created using webgl, javascript and html. 
It is composed of:
<ul>
  <li>Animation</li>
  <li>Sound</li>
  <li>Primitive object</li>
  <li>Surface of revolution</li>
  <li>Meshed Objects</li>
  <li>Texture for every object in the scene </li>
 </ul>
